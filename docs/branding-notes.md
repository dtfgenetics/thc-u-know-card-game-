# Branding Notes

THC U Know must be treated as an original parody-style game project.

## Do not use

- Official UNO logo.
- Official UNO card artwork.
- Official UNO card backs.
- Official UNO fonts or trade dress.
- Official UNO product photos.
- Official Mattel branding.

## Use instead

- Original THC U Know logo and card backs.
- Original cannabis parody card names.
- Original color/suit system.
- Original table layout.
- Original sound effects and music.
