# THC U Know Game Rules

THC U Know is an original UNNO-style parody card game. These rules describe the first playable digital version.

## Players

- 2-8 players.
- Each player chooses a display name before joining.
- The host creates a Smoke Circle and shares the invite link or Session Code.

## Objective

Be the first player to empty your hand.

## Starting the game

- Each player receives 7 cards by default.
- One non-wild card starts the Ashtray/discard pile.
- The first player in the Smoke Circle starts.

## Matching rules

A player may play a card if it matches:

- The active strain color.
- The number on the top Ashtray card.
- The action type on the top Ashtray card.
- Any wild card.

## Card names

| Card | Effect |
| --- | --- |
| Number card | Play by matching color or number. |
| Couch Lock | Skip the next player. |
| Puff Puff Pass Back | Reverse turn direction. Acts as skip in 2-player games. |
| Pack Two | Next player draws 2. |
| Strain Switch | Choose the active strain color. |
| Hotbox +4 | Choose active strain color; next player draws 4. |
| THC U Know! | One-card call when you are close to going out. |

## Party Mode cards

Party Mode adds chaos cards:

- Munchies
- Paranoia
- Rotation
- Tolerance Break
- Bogart
- Pass the Tray
- Smoke Sesh
- Greener Side

These are scaffolded in the deck and reducer. Their final balance should be tested before launch.

## Anti-cheat rule

The server is the source of truth. The browser may request a move, but the server decides whether it is valid.
